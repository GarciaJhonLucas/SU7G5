# Proyecto Unidad 7 - Grupo 5
API REST creando con las tecnologias mostradas

## Tecnologias
- Node
- Prisma
- Typescript
- NodeJs
- Express

## Modo de uso

- Cambiar la el nombre del archivo env a .env
- Luego ejecutar los siguientes comandos en su terminal

```bash
npm install
```
```bash
npx prisma db push
```
```bash
npx prisma db seed
```
```bash
npm run dev
```
